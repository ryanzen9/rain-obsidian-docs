和大家简单分享下我的一个使 UI 好看的小技巧。（当然啦，还有很多小技巧，时间来不及，一个一个发）

技巧之一：使用 Radix UI Colors 及其使用规则来进行页面 UI 设计。

Radix UI Colors 是我近两年最喜欢的颜色设计系统，相较于 TailwindCSS Colors，它的颜色系统是更全面，还支持透明变体，自动深色模式等。

当然，最硬核，或者说最适合非专业设计师（或独立开发者）的是，它有一套使用规范，12 条规则说明什么颜色用在什么场景，照着这个配置，基本就能搭配出不错的色彩配置。

最主要的是，完全解决了思考决策的负担，非常好用！
