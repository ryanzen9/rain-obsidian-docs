# MCP

## 什么是 MCP

**MCP（模型上下文协议，Model Context Protocol）**是一个开源标准，旨在将 AI 应用程序与外部系统安全地连接起来。你可以将 MCP 形象地比作 AI 应用的“USB-C 接口”——就像 USB-C 为各种电子设备提供了标准化的物理连接一样，MCP 为 AI 应用（如 Claude 或 ChatGPT）连接外部数据源（如本地文件、数据库）、工具（如计算器、搜索引擎）以及工作流提供了统一的协议标准。

简单来说，就是为 LLM 提供双手，使其可以和外界交互。

> 为什么搭建 Vibe Coding 架构需要它？ 因为让 LLM 与外界交互可以让他获取更准确，更充足的上下文。

MCP 采用的是经典的客户端-服务器（Client-Server）架构。它的整体架构主要由三大核心参与者和两大核心通信层组成。(详情见`https://modelcontextprotocol.io/docs/learn/architecture`)

## MCP 服务器的服务类型

### 1. 本地服务器 (Local MCP Servers)

- 特点：运行在用户同一台机器上的本地进程中
- 传输协议：使用 stdio（标准输入/输出） 进行通信，性能最佳且没有网络开销
- 适用场景：非常适合需要直接访问本地系统资源的工具，例如本地文件系统操作、本地数据库（如 PostgreSQL）或运行自定义脚本

### 2. 远程服务器 (Remote MCP Servers)

- 特点：运行在云平台或远程机器上
- 传输协议：主要推荐使用 Streamable HTTP 传输（支持使用 OAuth 等标准进行身份验证），早期也支持 SSE（Server-Sent Events，目前已弃用）
- 适用场景：适合连接基于云的服务和第三方 API，例如官方提供的 Sentry、Notion、GitHub 或 Slack 等 MCP 服务器

## 在 ClaudeCode 中使用 MCP

Claude Code 可以通过模型上下文协议（MCP）连接到数百个外部工具和数据源。MCP 服务器为 Claude Code 提供对您的工具、数据库和 API 的访问权限。

### MCP 的作用域

- 本地作用域 (Local scope)：配置保存在用户的本地目录`~/.claude`中，仅对当前项目的当前用户私有可见。
- 项目作用域 (Project scope)：配置保存在项目根目录的 `.mcp.json` 文件中，可以提交到版本控制系统中。非常适合团队协作，确保所有成员都拥有相同的工具
- 用户作用域 (User scope)：跨项目全局可用，但仅对当前用户的账号私有。适合个人常用的开发工具或实用程序

### 安装

共有三种安装方法，Http，SSE，STD，取决于 Transport 的不同，这里简单讲一下最通用的 Http。其他同理。

```bash
# Basic syntax
claude mcp add --transport http <name> <url>

# Real example: Connect to Notion
claude mcp add --transport http notion https://mcp.notion.com/mcp

# Example with Bearer token
claude mcp add --transport http secure-api https://api.example.com/mcp \
  --header "Authorization: Bearer your-token"
```

### 其他相关命令

```bash
# List all configured servers
claude mcp list

# Get details for a specific server
claude mcp get github

# Remove a server
claude mcp remove github

# (within Claude Code) Check server status
/mcp
```

## Example

### 为项目安装 Context7 MCP Server(最新的 LLM 和 AI 代码编辑器文档)

```bash
# Add a project-scoped server
claude mcp add --transport http context7 --scope project https://mcp.context7.com/mcp
```

```diff
# .mcp.json
+{
+  "mcpServers": {
+    "context7": {
+      "type": "http",
+      "url": "https://mcp.context7.com/mcp"
+    }
+  }
+}
```

### 使用 MCP 连接 SupaBase 作为基础数据库

```bash
claude mcp add --transport http supabase https://mcp.supabase.com/mcp
```

在 ClaudeCode 中执行 `/mcp` 进行认证。

```bash
> claude mcp list
context7: https://mcp.context7.com/mcp (HTTP) - ✓ Connected
supabase: https://mcp.supabase.com/mcp (HTTP) - ✓ Connected
```
